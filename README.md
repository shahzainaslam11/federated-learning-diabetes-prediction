# Explainable Federated Learning for Privacy-Preserving Diabetes Prediction

## Overview
This project presents a federated learning approach for predicting Type-2 diabetes risk while preserving data privacy. It enables collaborative model training across multiple institutions without sharing sensitive patient data.

## Key Contributions
- Federated learning framework for healthcare prediction  
- Integration of Differential Privacy for secure training  
- Handling class imbalance using SMOTE and SMOTE-ENN  
- Explainable AI (SHAP) for model interpretability  

## Methods
- **Framework:** Federated Learning (FedProx)  
- **Models:** Random Forest, XGBoost, LightGBM  
- **Privacy:** Gradient clipping and noise injection  
- **Explainability:** SHAP-based insights  

## Results
- Strong predictive performance in distributed settings  
- Preserved data privacy across multiple sources  
- Improved transparency for clinical decision-making  

## Publication
This work was peer-reviewed, accepted, presented, and published at **ICCES 2026 (ACM)**.

## Purpose
Supports the development of privacy-preserving and trustworthy AI systems in healthcare, enabling secure collaboration and informed clinical decisions.
